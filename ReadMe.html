Basic CRM App

This is client side chat application, which acheive the following requirements:

  1. The homepage shoud show all the existing users.
  2. All should be done by Redux lifecycle.
  3. Double clicking on each box will launch a modal that will allow the user to edit the information

The application demonstrate the following skills been used to develop this app:

  1. Implement end to end React/Redux flow.
  2. Simulate a REST service response which get the information about users.
  3. Buidling basic HTML pages for App, User and Form screens.
  4. Design the application and make it looks nice through CSS skills.
  5. Openning modal by each user double-click
  6. Submiting changes on cards after editing user properties.
